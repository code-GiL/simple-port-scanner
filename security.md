# Kebijakan Keamanan (Security Policy)

## Tujuan Edukasi
Proyek ini ditujukan semata-mata untuk **edukasi** guna mendemonstrasikan konsep dasar jaringan komputer dan pemrograman Python. Kode ini tidak dirancang untuk melewati kontrol keamanan canggih atau mengeksploitasi kerentanan.

## Melaporkan Masalah
Jika Anda menemukan *bug* atau celah keamanan pada alat ini, atau memiliki kekhawatiran tentang potensi penyalahgunaannya, harap hubungi pemilik repositori melalui email pribadi, jangan membukanya di *Public Issue*.

## Penggunaan Aman
Pengguna sangat disarankan untuk menjalankan alat ini hanya di lingkungan laboratorium terisolasi (seperti Virtual Machine atau LAN Rumah) untuk menghindari gangguan pada jaringan produksi.
